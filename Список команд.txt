Этот список необходимо скормить BotFather для нормльной работы

help - Вызвать помощь по системе
start - Начало работы с системой
allclear - Очистка списка адресов
listaddress - Список адресов пользователь
test - Сделать запрос по всем адресам для текущего пользователя
subscribe - Всключить рассылку
unsubscribe - выключить рассылку
